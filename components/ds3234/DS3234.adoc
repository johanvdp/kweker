== DS3234 module

*Module name: RTC*

Real-time clock module.
Communicates using SPI interface.

.DS3234 module
[%autowidth]
|===
| pin | function | description

| 1   | SS       | active low chip select
| 2   | MOSI     | SPI
| 3   | MISO     | SPI
| 4   | CLK      | SPI
| 5   | SQW      | unused
| 6   | VCC      |
| 7   | GND      |
|===
