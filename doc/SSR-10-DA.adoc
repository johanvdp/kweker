== SSR-10 DA

Solid state relay.
Used to switch AC loads on and off.

.SSR-10 DA module
[cols=1;2]
|===
| pin   | description

| *1*
.2+| AC switch
| *2*

| *3*   | DC control+
| *4*   | DC control-

|===

.SSR-10 DA module limits
[cols=1;1]
|===
| parameter | value

| DC control | 3..32V DC
| AC switch voltage | 24..380V AC
| AC switch current | < 10 A
|===

.SSR-10 DA module usage
[cols=1;3]
|===
| device | description

| SSR1 | Light
| SSR2 | Exhaust fan
| SSR3 | Recirculation fan
| SSR4 | Heater
|===
